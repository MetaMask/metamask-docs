- `address`: [_Required_] A string representing the address (20 bytes) of the smart contract, from which the compiled byte code will be obtained.
- `block number`:[_Required_] A hexadecimal block number, or one of the string tags `latest`, `earliest`, `pending`, `safe`, or `finalized`. See the [default block parameter](https://ethereum.org/en/developers/docs/apis/json-rpc/#default-block).
