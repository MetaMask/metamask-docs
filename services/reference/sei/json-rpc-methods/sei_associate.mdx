# `sei_associate`

Sends a transaction to link the signer's Sei address and EVM address onchain. <CreditCost network="evm" method="sei_associate" />

## Parameters

- CUSTOM MESSAGE OBJECT [_Required_]
  - `custom_message`: Any string message.
  - `r`: The R-part of the signature over the Keccak256 hash of the custom message.
  - `s`: The S-part of the signature over the Keccak256 hash of the custom message.
  - `v`: The V-part of the signature over the Keccak256 hash of the custom message.

## Returns

The transaction hash as a string.

## Example

Replace `<YOUR-API-KEY>` with an API key from your MetaMask Developer dashboard.

<Tabs>
  <TabItem value="curl">

```bash
curl https://sei-mainnet.infura.io/v3/<YOUR-API-KEY> \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "sei_associate",
    "params": [
      {
        "custom_message": "Associate my addresses",
        "v": "0x1c",
        "r": "0x88ff6cf0fefd94db46111149ae4bfc179e9b94721fffd821d38d16464b3f71d0",
        "s": "0x45e0aff800961cfce805daef7016b9b675c137a6a41a548f7b60a3484c06a33a"
      }
    ],
    "id": 1
  }'
```

  </TabItem>
</Tabs>
