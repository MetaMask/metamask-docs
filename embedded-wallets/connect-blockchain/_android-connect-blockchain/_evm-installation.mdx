To interact with the Ethereum Compatible Blockchain in Android, you can use any
[`EIP1193`](https://eips.ethereum.org/EIPS/eip-1193) compatible package available for Android. Here
we're using [web3j](https://github.com/web3j/web3j) to demonstrate how to make blockchain calls
using it with Web3Auth.

```groovy
dependencies {
  // ...
  implementation 'org.web3j:core:4.8.7-android'
}
```
