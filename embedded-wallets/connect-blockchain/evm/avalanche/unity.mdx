While using the Embedded Wallets Unity SDK (formerly Web3Auth), you get the private key within the user scope. Use it with Nethereum to make EVM-based blockchain calls on [Avalanche (C-Chain)](https://www.avax.network/): get account, fetch balance, sign and send transactions, and interact with contracts.

::::note

The SDKs are now branded as MetaMask Embedded Wallet SDKs (formerly Web3Auth Plug and Play SDKs). Package names and APIs remain Web3Auth (for example, Web3Auth React SDK), and code snippets may reference `web3auth` identifiers.

::::
