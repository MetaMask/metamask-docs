While using the Embedded Wallets Unity SDK (formerly Web3Auth), you get the private key within the user scope. This private key can interact with the [Nethereum Library](https://nethereum.com/) to make EVM-based blockchain calls, like getting the user's `account`, fetching `balance`, `signing transaction`, `sending transactions`, and interacting with smart contracts. We have highlighted a few examples to get you started.
