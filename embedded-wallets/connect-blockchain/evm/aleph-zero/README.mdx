The Aleph Zero blockchain's integration with Embedded Wallets (formerly Web3Auth) helps you get started in minutes. The integration is available for the following platforms: Web, Android, iOS, Flutter, React Native, and Unity.

::::note

The SDKs are now branded as MetaMask Embedded Wallet SDKs (formerly Web3Auth Plug and Play SDKs). Package names and APIs remain Web3Auth (for example, Web3Auth React SDK), and code snippets may reference `web3auth` identifiers.

::::
