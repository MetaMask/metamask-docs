The Ancient8 blockchain's integration with Embedded Wallets (formerly Web3Auth) helps you get started in minutes. The integration is available for Web and Embedded Wallets Mobile SDKs (Android, iOS, React Native), as well as Flutter, React Native, and Unity.

::::note

The SDKs are now branded as MetaMask Embedded Wallet SDKs (formerly Web3Auth Plug and Play SDKs). Package names and APIs remain Web3Auth (for example, Web3Auth React SDK), and code snippets may reference `web3auth` identifiers.

::::
