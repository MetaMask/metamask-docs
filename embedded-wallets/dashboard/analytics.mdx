## Metrics

### Total logins

The cumulative number of login events recorded for the project. Use this to track overall activity and spot spikes or drops in authentication frequency.

### Monthly active wallets

Monthly Active Wallets (MAWs) counts unique wallets that authenticated at least once in a calendar month. Use this to track user retention over time.

### Monthly active wallets overview

A pie chart breaking down MAWs by wallet type or authentication method. Use this to understand which login methods your users prefer.

### Connections used

A list of the authentication connection types used across the project, such as social logins, email passwordless, and external wallets.
