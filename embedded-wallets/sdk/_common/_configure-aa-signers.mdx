The Web3Auth Smart Account feature is compatible with popular signer SDKs, including Wagmi, ethers,
and viem. You can choose your preferred package to configure the signer.

<TabItem value="wagmi">

Use Wagmi with `@web3auth/modal/react/wagmi` or `@web3auth/modal/vue/wagmi`. After sign-in, Wagmi
hooks (`useAccount`, `useSendTransaction`, and similar) use the smart account signer automatically.
No manual `connection.ethereumProvider` wiring is required.

```tsx
import { useAccount, useSendTransaction } from '@web3auth/modal/react/wagmi'

const { address } = useAccount()
// useSendTransaction, useWriteContract, etc. work with the smart account
```

</TabItem>

<TabItem value="viem">

For vanilla JavaScript or apps without Wagmi, read the EIP-1193 provider from `connection`:

```tsx
import { createWalletClient, custom } from 'viem'

const connection = web3auth.connection
const provider = connection?.ethereumProvider ?? null

const walletClient = createWalletClient({
  transport: custom(provider),
})
```

</TabItem>

<TabItem value="ethers">

```tsx
import { ethers } from 'ethers'

const connection = web3auth.connection
const provider = connection?.ethereumProvider ?? null

const ethersProvider = new ethers.providers.Web3Provider(provider)
const signer = await ethersProvider.getSigner()
```
