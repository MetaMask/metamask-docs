To retrieve the secp256k1 private key of the user, you can access it from the login response. The method returns an EVM compatible private key which can be used to sign transactions on EVM compatible chains.

## Usage

```cs
private void onLogin(Web3AuthResponse response)
{
    var privateKey = response.privKey;
    Debug.Log("Secp256k1 Private Key: " + privateKey);

    // Use private key for EVM transactions
}
```
