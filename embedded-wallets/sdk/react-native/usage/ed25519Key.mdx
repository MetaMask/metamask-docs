Using the `ed25519Key()` method in the Web3Auth instance you can get the `ed25519` private key.

## Method

`get ed25519Key(): string;`

## Usage

```tsx
const ed25519Key = web3auth.ed25519Key()
```

## Response

```string
"666523652352635...."
```
