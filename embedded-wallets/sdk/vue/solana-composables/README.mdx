Embedded Wallets provides a set of Vue composables for basic Solana wallet operations. These composables are designed to simplify common Solana interactions in your Vue app.

:::info

For building transactions, use [`@solana/kit`](https://www.npmjs.com/package/@solana/kit) and [`@solana-program/system`](https://www.npmjs.com/package/@solana-program/system) with the composables provided.

:::
