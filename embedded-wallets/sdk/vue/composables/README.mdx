Embedded Wallets offers a comprehensive suite of Vue composables to streamline integration and state management within your Vue applications. These composables abstract away common authentication and wallet operations, enabling you to implement Embedded Wallets features with minimal boilerplate. Each composable is focused on a specific aspect of Embedded Wallets functionality, making it easy to implement authentication, wallet operations, and blockchain interactions in your Vue projects.

## List of Vue composables

:::tip

For detailed usage, configuration options, and code examples, refer to the dedicated documentation page for each composable.

:::

### Embedded Wallets composables

| Composable Name | Description                                                       |
| --------------- | ----------------------------------------------------------------- |
| [`useWeb3Auth` ](./useWeb3Auth.mdx)  | Core composable for Embedded Wallets/Web3Auth initialization and state management. |

### Authentication composables

| Composable Name         | Description                            |
| ----------------------- | -------------------------------------- |
| [`useWeb3AuthConnect`](./useWeb3AuthConnect.mdx)       | Handles Embedded Wallets/Web3Auth connection processes. |
| [`useWeb3AuthDisconnect`](./useWeb3AuthDisconnect.mdx) | Manages disconnection from Embedded Wallets/Web3Auth.   |

### Identity composables

| Composable Name    | Description                                              					      |
| ------------------ | -----------------------------------------------------------------------------------|
| [`useIdentityToken`](./useIdentityToken.mdx) | Retrieves and manages identity tokens.                   |
| [`useWeb3AuthUser`](./useWeb3AuthUser.mdx)   | Provides access to the authenticated user's information. |

### Blockchain composables

| Composable Name  | Description                                    						         |
| ---------------- | ------------------------------------------------------------------------------- |
| [`useSwitchChain`](./useSwitchChain.mdx) | Allows switching between different blockchain networks. |

### MFA composables

| Composable Name | Description                                             					           |
| --------------- | -------------------------------------------------------------------------------------- |
| [`useEnableMFA`](./useEnableMFA.mdx)  | Enables Multi-Factor Authentication (MFA) for enhanced security. |
| [`useManageMFA`](./useManageMFA.mdx)  | Provides functionality to manage MFA settings.                   |

### Wallet Services plugin composables

| Composable Name           | Description                                      							       |
| ------------------------- | -------------------------------------------------------------------------------- |
| [`useWalletConnectScanner`](./useWalletConnectScanner.mdx) | Integrates WalletConnect scanner functionality. |
| [`useWalletUI`](./useWalletUI.mdx)       			         | Manages wallet UI components and user interactions. |
| [`useCheckout` ](./useCheckout.mdx)          				 | Facilitates cryptocurrency checkout processes. |
| [`useSwap` ](./useSwap.mdx)                			  	 | Enables token swapping capabilities.  |
| [`useWalletServicesPlugin`](./useWalletServicesPlugin.mdx) | Provides access to wallet service plugins. |
