Embedded Wallets offers a comprehensive suite of React hooks to streamline integration and state management within your React applications. These hooks abstract away common authentication and wallet operations, enabling you to implement Embedded Wallets' features with minimal boilerplate. Each hook is focused on a specific aspect of Embedded Wallets' functionality, making it easy to implement authentication, wallet operations, and blockchain interactions in your React projects.

## List of React hooks

:::tip

For detailed usage, configuration options, and code examples, refer to the dedicated documentation page for each hook.

:::

### Web3Auth hooks

| Hook Name   					    | Description                                      |
| --------------------------------- | ------------------------------------------------ |
| [`useWeb3Auth`](./useWeb3Auth.mdx) | Core hook for Embedded Wallets initialization and state management. |

### Authentication hooks

| Hook Name               								 | Description                            |
| ------------------------------------------------------ | -------------------------------------- |
| [`useWeb3AuthConnect`](./useWeb3AuthConnect.mdx)       | Handles Embedded Wallets connection processes. |
| [`useWeb3AuthDisconnect`](./useWeb3AuthDisconnect.mdx) | Manages disconnection from Embedded Wallets.   |

### Identity hooks

| Hook Name  						           | Description                              |
| -------------------------------------------- | ---------------------------------------- |
| [`useIdentityToken`](./useIdentityToken.mdx) | Retrieves and manages identity tokens.   |
| [`useWeb3AuthUser`](./useWeb3AuthUser.mdx)   | Provides access to the authenticated user's information. |

### Blockchain hooks

| Hook Name      						  | Description                                  |
| ---------------------------------------- | ------------------------------------------- |
| [`useSwitchChain`](./useSwitchChain.mdx) | Allows switching between different blockchain networks. |

### Multi-Factor Authentication hooks

| Hook Name      | Description                                                      |
| -------------- | ---------------------------------------------------------------- |
| [`useEnableMFA`](./useEnableMFA.mdx) | Enables Multi-Factor Authentication (MFA) for enhanced security. |
| [`useManageMFA`](./useManageMFA.mdx) | Provides functionality to manage MFA settings.                   |

### Wallet Services plugin hooks

| Hook Name           								         | Description    				|
| ---------------------------------------------------------- | ---------------------------- |
| [`useWalletConnectScanner`](./useWalletConnectScanner.mdx) | Integrates WalletConnect scanner functionality.     |
| [`useWalletUI`](./useWalletUI.mdx)            			 | Manages wallet UI components and user interactions. |
| [`useCheckout`](./useCheckout.mdx)            			 | Facilitates cryptocurrency checkout processes. |
| [`useSwap`](./useSwap.mdx)                 				 | Enables token swapping capabilities. |
| [`useWalletServicesPlugin`](./useWalletServicesPlugin.mdx) | Provides access to Wallet Services plugins. |
