The following table displays the Advanced Permissions types supported by the 
Smart Accounts Kit, [MetaMask Flask](/snaps/get-started/install-flask), and MetaMask production, and the minimum version required for each.

If you don't see the Advanced Permissions type you're looking for, you can request it by 
[emailing support](mailto:hellogators@consensys.net).

| Permission type                                                                            | Smart Accounts Kit | MetaMask Flask | MetaMask |
| ------------------------------------------------------------------------------------------ | ------------------ | -------------- | -------- |
| [ERC-20 periodic](../guides/advanced-permissions/use-permissions/erc20-token.md#erc-20-periodic-permission)         | >= v0.1.0          | >= v13.5.0     | N/A        |
| [ERC-20 stream](../guides/advanced-permissions/use-permissions/erc20-token.md#erc-20-stream-permission)             | >= v0.1.0          | >= v13.5.0     | N/A        |
| [ERC-20 revocation](../guides/advanced-permissions/use-permissions/erc20-token.md#erc-20-revocation-permission)     | >= v0.3.0          | >= v13.14.0    | N/A        |
| [Native token periodic](../guides/advanced-permissions/use-permissions/native-token.md#native-token-periodic-permission) | >= v0.1.0          | >= v13.5.0     | N/A        |
| [Native token stream](../guides/advanced-permissions/use-permissions/native-token.md#native-token-stream-permission)     | >= v0.1.0          | >= v13.5.0     | N/A        |
