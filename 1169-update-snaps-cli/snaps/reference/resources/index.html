Resources

View the following Snaps resources in addition to this documentation site.

Primary resources​

  • Snaps homepage
  • Snaps directory - A directory of allowlisted Snaps you can try.
  • SIPs - Snaps Improvement Proposals.
  • Open Beta Readiness Guide - A guide to get your Snap allowlisted on MetaMask.

Example Snaps​

  • Example Snaps - A directory of Snaps maintained by MetaMask.
  • Dogecoin Snap - A tutorial for creating a Snap to add Dogecoin to MetaMask.
  • Starknet Snap - A Snap to add Starknet to MetaMask.
  • MobyMask Phishing Warning Snap - A Snap that warns you when interacting with phishing contracts.
  • Mystery Snap - A Snap that returns random numbers and answers.
  • Snappy Recovery - A Snap that enables social recovery of private keys.
  • Pet Fox game - A simple game built using a Snap.

Developer tools​

  • Template Snap - A rich template that includes TypeScript/React and vanilla JavaScript options, a CLI for building, packaging, and deploying your Snap, and a companion dapp UI you can build on.
  • Snaps Simulator - A tool for simulating Snaps in the browser, streamlining the development process.
  • Test Snaps - A collection of test Snaps and a dapp for evaluating them.
