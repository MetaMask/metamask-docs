# JSON-RPC API

This section provides an interactive reference for MetaMask's JSON-RPC API.
The API builds on a set of standard Ethereum methods with MetaMask-specific enhancements, and is designed for seamless integration into dapps.
View the JSON-RPC API methods by selecting a method in the left sidebar.
