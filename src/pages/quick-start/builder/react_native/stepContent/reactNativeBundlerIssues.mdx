---
title: Fixing Bundler Issues
---

While using Web3Auth in React Native, you may run into issues building. This issue occurs because
some core packages like `eccrypto` have certain dependencies which are not present within the Metro
build environment.

:::tip Troubleshooting

To solve this, please have a look at our troubleshooting page:

- [React Native Metro Bundler Issues](/troubleshooting/metro-issues)

:::
