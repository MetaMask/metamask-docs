### Make blockchain calls

Use Wagmi composables to make blockchain calls.
This quickstart uses `useBalance`, `useAccount`, `useSignMessage`, `useSendTransaction`, and `useSwitchChain`.

See the [Wagmi documentation](https://wagmi.sh/vue/api/composables) for more information on the composables.

:::tip Connect any blockchain

See the [Embedded Wallets documentation](/embedded-wallets/connect-blockchain)
to learn more about making calls on any supported blockchain.

:::
