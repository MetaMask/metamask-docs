### Set up the Wagmi provider

In `App.vue`, nest `WagmiProvider` from `@web3auth/modal/vue/wagmi` inside `Web3AuthProvider`.
Register `VueQueryPlugin` in `main.ts` before mounting the app.

```ts
import { WagmiProvider } from '@web3auth/modal/vue/wagmi'
```

You do not need a separate Wagmi config file; the Web3Auth provider wires the connector.
See the [Wagmi documentation](https://wagmi.sh/vue/getting-started) for more information.
