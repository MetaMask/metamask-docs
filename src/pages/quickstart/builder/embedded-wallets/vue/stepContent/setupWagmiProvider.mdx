### Set up the Wagmi provider

Since this quickstart uses Wagmi, wrap your app with the `WagmiProvider` from
`@web3auth/modal/vue/wagmi`.

```tsx
import { WagmiProvider } from '@web3auth/modal/vue/wagmi'
```

This quickstart uses the standard Wagmi setup.
You don't need to create a Wagmi configuration; the Web3Auth configuration is used directly.
See the [Wagmi documentation](https://wagmi.sh/vue/getting-started) for more information.
