### Troubleshoot bundler issues

You might see build or bundle errors when you use MetaMask Embedded Wallets in Vue.
Some packages (for example, `eccrypto`) depend on Node.js modules that aren't available in the browser build environment.

See [Bundler Polyfill issues](/embedded-wallets/troubleshooting/vite-issues) for more information.

:::info

If you need help, visit the [MetaMask Builder Hub](https://builder.metamask.io/).

:::
