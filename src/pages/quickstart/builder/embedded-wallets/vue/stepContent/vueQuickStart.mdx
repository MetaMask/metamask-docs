### MetaMask Embedded Wallets Vue quickstart

This quickstart shows you how to add social sign-in and wallet connection functionality to your Vue application,
using the MetaMask Embedded Wallets SDK (Web3Auth).

Clone the Vue quickstart application to follow along with a complete example:

```shell
npx degit Web3Auth/web3auth-examples/quick-starts/vue-quick-start w3a-quick-start
```
