### Sign out the user

Use the [`useWeb3AuthDisconnect`](/embedded-wallets/sdk/react/hooks/useWeb3AuthDisconnect) hook to
sign out the user. This also clears the session information from the browser's local storage.

:::info

The Embedded Wallets React SDK provides additional hooks.
See the [hooks reference](/embedded-wallets/sdk/react/hooks) for more information.

:::
