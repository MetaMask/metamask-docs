### MetaMask Embedded Wallets React quickstart

This quickstart shows you how to add social sign-in and wallet connection functionality to your React application,
using the MetaMask Embedded Wallets SDK (Web3Auth).

:::note
To view the quickstart for a different platform (for example, Next.js, Android, or iOS),
select the dropdown in the menu bar and choose your target platform.
:::

Clone the React quickstart application to follow along with a complete example:

```bash
npx degit Web3Auth/web3auth-examples/quick-starts/react-quick-start w3a-quick-start
```
