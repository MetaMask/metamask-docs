### Troubleshoot bundler issues (if needed)

`@web3auth/modal@11` runs on Vite without extra polyfills in most projects. Build your app first;
only add polyfills if the bundler reports missing Node.js modules.

See [Bundler polyfill issues](/embedded-wallets/troubleshooting/vite-issues/) for fallback steps.

:::info

If you need help, visit the [MetaMask Builder Hub](https://builder.metamask.io/).

:::
