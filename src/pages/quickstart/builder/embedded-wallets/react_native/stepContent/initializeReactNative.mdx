### Initialize Web3Auth

Initialize a `Web3Auth` instance with your project's client ID and your target network
(this quickstart uses `SAPPHIRE_MAINNET`).
Add a `redirectUrl` for your application, so the SDK can redirect users back to your app after authentication.

:::info

See the [Embedded Wallets React Native SDK documentation](/embedded-wallets/sdk/react-native) for more information.

:::
