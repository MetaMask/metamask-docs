### Install the Embedded Wallets SDK

Install the MetaMask Embedded Wallets React Native SDK and its helper packages:

```bash
npm install --save @web3auth/react-native-sdk @toruslabs/react-native-web-browser react-native-encrypted-storage
npm install --save-dev @babel/plugin-transform-export-namespace-from
```
