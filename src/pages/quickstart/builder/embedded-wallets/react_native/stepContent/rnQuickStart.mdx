### MetaMask Embedded Wallets React Native quickstart

This quickstart shows you how to add social sign-in and wallet connection functionality to your React Native application,
using the MetaMask Embedded Wallets SDK (Web3Auth).

Clone the React Native quickstart application to follow along with a complete example:

```bash
npx degit Web3Auth/web3auth-react-native-examples/rn-bare-quick-start w3a-quick-start
```

:::warning

The MetaMask Embedded Wallets React Native SDK is not compatible with Expo Go.
Use an Expo Custom Dev Client or an EAS build instead.
See [Bundler polyfill issues](embedded-wallets/troubleshooting/metro-issues)
for more information.

:::
