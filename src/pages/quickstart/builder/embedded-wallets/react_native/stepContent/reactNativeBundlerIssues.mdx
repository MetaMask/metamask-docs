### Troubleshoot bundler issues

You might see build or bundle errors when you use MetaMask Embedded Wallets in React Native.
Some packages (for example, `eccrypto`) depend on modules that Metro doesn't provide.

See [Bundler Polyfill issues](/embedded-wallets/troubleshooting/metro-issues) for more information.

:::info

If you need help, visit the [MetaMask Builder Hub](https://builder.metamask.io/).

:::
