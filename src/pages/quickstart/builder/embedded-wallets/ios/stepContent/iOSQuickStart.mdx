### MetaMask Embedded Wallets iOS/Swift quickstart

This quickstart shows you how to add social sign-in and wallet connection functionality to your iOS or Swift application,
using the MetaMask Embedded Wallets SDK (Web3Auth).

Clone the iOS/Swift quickstart application to follow along with a complete example.
You can [build and run](https://developer.apple.com/documentation/xcode/building-and-running-an-app) the example in Xcode.

```bash
npx degit Web3Auth/web3auth-ios-examples/ios-quick-start w3a-ios-quick-start
```
