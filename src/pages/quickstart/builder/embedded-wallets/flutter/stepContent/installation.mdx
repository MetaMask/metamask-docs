### Install the Embedded Wallets SDK

Install the MetaMask Embedded Wallets (Web3Auth) [Flutter SDK](https://github.com/Web3Auth/web3auth-flutter-sdk).
Add `web3auth_flutter` as a dependency to your `pubspec.yaml` file:

```sh
flutter pub add web3auth_flutter
```

or

```yaml
dependencies:
  web3auth_flutter: ^3.1.1 ### or the latest version
```

:::warning Update to the latest SDK

View the release notes to ensure you're using the [latest version](https://github.com/Web3Auth/web3auth-flutter-sdk/releases) of
the Flutter SDK.

:::
