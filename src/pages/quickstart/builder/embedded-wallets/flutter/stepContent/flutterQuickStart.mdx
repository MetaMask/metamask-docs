### MetaMask Embedded Wallets Flutter quickstart

This quickstart shows you how to add social sign-in and wallet connection functionality to your Flutter application,
using the MetaMask Embedded Wallets SDK (Web3Auth).

Clone the Flutter quickstart application to follow along with a complete example.
You can run the example in Android Studio or VS Code.

```bash
npx degit Web3Auth/web3auth-flutter-examples/flutter-quick-start flutter-quick-start
```
