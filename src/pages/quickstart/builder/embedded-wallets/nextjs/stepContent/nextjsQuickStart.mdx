### MetaMask Embedded Wallets Next.js quickstart

This quickstart shows you how to add social sign-in and wallet connection functionality to your Next.js application,
using the MetaMask Embedded Wallets SDK (Web3Auth).

Clone the Next.js quickstart application to follow along with a complete example:

```shell
npx degit Web3Auth/web3auth-examples/quick-starts/nextjs-quick-start w3a-quick-start
```
