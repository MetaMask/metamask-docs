### MetaMask Embedded Wallets Angular quickstart

This quickstart shows you how to add social sign-in and wallet connection functionality to your Angular application,
using the MetaMask Embedded Wallets SDK (Web3Auth).

Clone the Angular quickstart application to follow along with a complete example:

```bash
npx degit Web3Auth/web3auth-examples/quick-starts/angular-quick-start w3a-quick-start
```
