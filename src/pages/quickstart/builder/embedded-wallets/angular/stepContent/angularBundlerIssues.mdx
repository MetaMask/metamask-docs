### Troubleshoot bundler issues (if needed)

`@web3auth/modal@11` on Angular needs only `zone.js` in `polyfills.ts`. Do not add `Buffer`,
`process`, or `global` shims unless an older dependency still requires them.

See [Bundler polyfill issues](/embedded-wallets/troubleshooting/webpack-issues/) for legacy
fallback steps.

:::info

If you need help, visit the [MetaMask Builder Hub](https://builder.metamask.io/).

:::
