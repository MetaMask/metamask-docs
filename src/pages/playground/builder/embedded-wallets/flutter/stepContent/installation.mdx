---
title: Install Web3Auth
---

For making hybrid iOS & Android applications using Flutter, we have the
[web3auth-flutter-sdk](https://github.com/Web3Auth/web3auth-flutter-sdk) to integrate Web3Auth into
your application.

Add `web3auth_flutter` as a dependency to your `pubspec.yaml` file.

```sh
flutter pub add web3auth_flutter
```

or

```yaml
dependencies:
  web3auth_flutter: ^3.1.1 # or the latest version
```

:::warning Update to the Latest-SDK

Please use the **[latest version](https://github.com/Web3Auth/web3auth-flutter-sdk/releases)** of
Web3Auth's Flutter SDK.

:::
